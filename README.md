# NEWorld

NEWorld是一个开放源代码的模仿Minecraft游戏规则的游戏。

NEWorld采用[GPLv3许可证](http://www.gnu.org/licenses/gpl.html)发布并受其保护。

如果您想加入开发，请在SIGNOFF文件中署名。
